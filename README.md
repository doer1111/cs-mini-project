# CS Mini Project - Multi-Tool CLI App

## Overview
This is a simple Python command-line application built as a learning project.

It contains multiple tools inside a menu-driven system.

## Features
- Number Analyzer:
  - Even / Odd detection
  - Positive / Negative / Zero detection
  - Square calculation

- Calculator:
  - Addition
  - Subtraction
  - Multiplication
  - Division (with zero-check)

- Menu system:
  - Loop-based navigation
  - Exit option

## How to Run
```bash
python3 main.py

# CS Mini Project - Multi-Tool CLI App

## Overview
This is a beginner Python command-line application built as a learning project.

It contains multiple tools inside a menu-driven system.

---

## Features

### Number Analyzer
- Even / Odd detection
- Positive / Negative / Zero detection
- Square calculation

### Calculator
- Addition
- Subtraction
- Multiplication
- Division (with zero-check)

### Guessing Game
- 3 difficulty modes (Easy, Medium, Hard)
- Hot / Warm / Cold hint system
- Score tracking system
- Persistent high score using JSON file storage

### Menu System
- Loop-based navigation using `while True`
- Exit option to terminate program

---

## Concepts Used
- Python functions
- Conditionals
- Loops
- Error handling
- File handling
- JSON storage
- Random number generation
